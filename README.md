# Sistem Pemesanan Barang
Aplikasi ini digunakan untuk mengelola data pemesanan barang berbasis web menggunakan Flask dan SQLite.
## Fitur
- Input data pemesanan
- Menampilkan data pemesanan
- Menyimpan data ke database
- Tampilan berbasis web

  
## Cara Menjalankan

1. Install Python
2. Install Flask
3. Jalankan file init_db.py
4. Jalankan app.py
5. Buka browser dan akses:
http://127.0.0.1:5000
